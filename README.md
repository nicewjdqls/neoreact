# Neo React (auto-converted)
- Pages generated from your HTML files as React components in `src/pages`.
- Routing set up in `src/App.jsx` (react-router-dom v6).
- Entry at `src/main.jsx`; HTML shell in `public/index.html`.

## Run (with Node 18+)
```bash
npm i
npm run dev
```
Then open the localhost URL.
